Data UI Table Catalog

The Data UI table catalog is a powerful and interactive interface designed to help users explore and analyze data effectively. Below is an expanded description of its features when no table rows are selected:

2. Interactive Table

The table is built using the Tabulator widget from the panel library, offering features like:

  • Sorting: Users can sort data by clicking on column headers.
  • Filtering: Header filters allow users to refine data based on specific criteria.
  • Pagination: Large datasets are divided into pages for better navigation.

3. Action Panel

An action panel is displayed above the table, containing buttons for various actions:

  • Plot: Visualize selected data in interactive plots.
  • Download: Export data as CSV files.
  • Permalink: Generate shareable links for the current view.

4. Integration with Maps

If geolocation data is available, the table integrates seamlessly with a map view:

  • Synchronization: Selecting rows in the table highlights corresponding features on the map and vice versa.
  • Interactive Map: Users can click on map features to view associated data in the table.

5. Customizable Options

Users can customize the table and map view using options provided in the sidebar:

  • Map Color Category: Change the color scheme for map features.
  • Point Size: Adjust the size of points on the map.

6. Progress Indicators

A progress bar provides feedback during data loading or processing tasks, ensuring users are informed about the status of their actions.

Start by selecting rows from the table to unlock these features and explore the data effectively.
